Starter kit Next.js app with TypeScript, Prisma, and tRPC
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Stack

- [Next.js](https://nextjs.org)
- [tRPC](https://nextjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Prisma](https://www.prisma.io)
